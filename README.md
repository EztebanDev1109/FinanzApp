# 💰 FinanzApp

Aplicación web para gestionar finanzas personales de forma simple, rápida y visual.  
Permite registrar ingresos, gastos, categorías y llevar control del dinero desde el navegador, porque sufrir con cuentas desordenadas ya era suficiente.

---

## 🚀 Demo

🔗 **Vista previa en línea:**

---

## ✨ Características

- 📊 Panel principal con resumen financiero
- 💵 Registro de ingresos y gastos
- 🏷️ Categorías personalizadas
- 📅 Historial de movimientos
- 💾 Almacenamiento en `localStorage`
- 📱 Diseño responsive
- 🎨 Interfaz moderna y limpia
- ⚡ Uso rápido y sin instalación

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
